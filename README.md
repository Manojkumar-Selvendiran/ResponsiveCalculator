# 🧮 Responsive Calculator (HTML, CSS, JavaScript)

This is a fully functional responsive calculator built using HTML, CSS, and JavaScript.

It supports both **keyboard input and button clicks**, and can evaluate full mathematical expressions like a real calculator.

---

## 🚀 Features

- Basic math operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Modulus (%)

- Full expression support  
  Example:  
  `12 + 5 * 3 - 4`

- Keyboard support:
  - Numbers (0–9)
  - Operators (+, -, *, /, %)
  - Backspace (delete last digit)
  - Enter (calculate result)

- Button click support for all operations

- Responsive design:
  - Works on desktop
  - Works on tablet
  - Works on mobile

- Clean and simple UI design

---

## 🛠️ Technologies Used

- HTML
- CSS (Flexbox and Media Queries)
- JavaScript (DOM manipulation, events, logic handling)

---

## 📂 Project Files

- index.html → Structure of calculator
- style.css → Styling and responsive design
- script.js → Calculator logic and functionality

---

## ⚙️ How It Works

- User enters input using keyboard or buttons
- Input is shown in a display screen
- Expression is processed using JavaScript logic
- Each mathematical operation has its own function:
  - add()
  - subtract()
  - multiply()
  - divide()
  - modulus()

- Expression is evaluated with proper operator precedence
- Result is displayed on screen
